cmd_add_machine

cmd_add_machine

描述

该主题包含控制端对后台下达的“添加机器”指令。后台收到该指令后，会登记此机器id，并订阅相关主题。后台通过cmd_ret主题返回是否成功。

消息实例

{ "cmd_id": "7a4Z2piq", "machine_id": "5Q4D1qwe" }

字段

cmd_id：指令id，每次调用指令都应当使用不同的id，以便在返回时区分是哪条指令的返回值。

machine_id：欲添加机器的id。
